# UML
统一建模语言：基于面相对象的可视化建模语言

# UML图例
1. 静态模型图：描述系统的静态结构
   1. 类图
   2. 对象图
   3. 包图
   4. 组件图
   5. 部署图
2. 动态模型图：描述系统行为的各个方面
   1. **用例图**
      用例图包括三个基本组件：参与者，用例，关系
      * 参与者：与系统打交道的人或其他系统即**使用该系统的人或事物**
      * 用例： 代表系统的某项完整的功能
      * 关系： 定义用例之间的关系
   3. 时序图
   4. 协作图
   5. 状态图
   6. 活动图
