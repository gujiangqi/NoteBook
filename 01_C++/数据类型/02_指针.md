# 指针定义
指针只存放变量的==首地址==
举例：
int a = 68;则系统会在内存中申请一个4个字节的地址，而申请一个指针变量 int *p = &a,则p中存放的仅仅是==变量的首地址仅占1个字节==，但是指针的内存大小通常是4个字节，而如何访问a呢？系统通过首地址查到这块内存，然后根据变量类型，向后查询4个字节，获取数据，所以必须定义指针类型，指针类型定义了向后访问几个字节。
```C++
    short a[2];
    a[0] = 1;
    a[1] = 1;

    short *p1 = a;
    int *p2 = (int *)p1;
   
    cout << *p1 << endl;
    cout << *p2 << endl;

```
