# ps
ps显示当前系统下的所有运行程序的执行状态，其中还包含了运行在该系统下的其他用户的进程信息
## 问题
==显示中的英文==是什么意思
## -a
用来显示同一终端下的所有程序
```
// open-ies01:/home/d5000/ies % ps -a
  PID TTY          TIME CMD
 1168 pts/5    00:00:00 ps
11638 pts/1    00:00:02 dbi
```
## -A
显示所有的进程
